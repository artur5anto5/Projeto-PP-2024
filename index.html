FETEGRASIM

A simulação digital de sistemas gravitacionais de n-corpos em três dimensões é essencial para entender a dinâmica dos sistemas presentes no universo. Porém, é uma tarefa complexa que requer alta capacidade computacional e gerenciamento de grandes volumes de dados.


A implementação de estruturas de dados adequadas é uma forma eficaz de aumentar a eficiência das simulações. Este projeto visa analisar o desempenho de diferentes estruturas de dados em simulações gravitacionais digitais, com o objetivo de identificar as estruturas mais eficientes.
